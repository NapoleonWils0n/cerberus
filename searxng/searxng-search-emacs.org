#+TITLE: searxng search
#+OPTIONS: toc:nil num:nil
#+STARTUP: overview hideblocks 
#+SELECT_TAGS: export
#+PROPERTY: header-args :results output wrap append :noweb yes :var page=1 query="nixos"
* usage
** global search

edit the PROPERTY at the top of the file and change the query and page number

#+begin_example
#+PROPERTY: header-args :results output wrap append :noweb yes :var page=1 query="buster keaton"
#+end_example

** org refesh

To tell Org-mode to re-scan the entire file for changes to #+PROPERTY put your cursor on the line and press:

#+begin_example
C-c C-c
#+end_example

** export results

1) Move your cursor so it is physically on the * results headline

2) Press C-c C-e to open the export settings
  
3) Press C-s this will toggle the setting from Buffer to Subtree

4) Once it says Subtree, press your export keys (e.g., m m for Markdown).

* noweb code blocks
** searxng-curl noweb
:CODE:
#+NAME: searxng-curl
#+begin_src sh 
curl -s -G "http://localhost:8080/search" \
--data-urlencode "q=${query}" \
--data "format=json" \
--data-urlencode "categories=${category}" \
--data "pageno=${page}" \
--data "language=auto" \
--data "time_range=" \
--data "safesearch=0"
#+end_src
:END:

** map noweb
:CODE:
#+NAME: map-noweb
#+begin_src sh :noweb yes
<<searxng-curl>> | jq -rj '.results | sort_by(.score) | reverse | to_entries[] | 
"\(.key + 1): [[\(.value.url)][\(.value.title)]]\n" + 
(if (.value.content and .value.content != "") then "\n\(.value.content)\n\n" else "\n" end) +
(if .value.address then 
  ([.value.address.house_number, .value.address.road, .value.address.locality, .value.address.postcode, .value.address.country] | del(..|nulls) | join(", ")) as $addr |
  (if ($addr | length) > 0 then "Address: \($addr)\n" else "" end)
else "" end) +
(if (.value.data and (.value.data | length) > 0) then 
  (.value.data | map(if .label == "open days" then "opening hours: \(.value)" else "\(.label): \(.value)" end) | join("\n")) + "\n" 
else "" end) +
(if .value.latitude then "Coords: \(.value.latitude), \(.value.longitude)\n" else "" end) +
(if .value.osm then "OSM: \(.value.osm.type)/\(.value.osm.id)\n" else "" end) +
"Score: \(.value.score)\nCategory: \(.value.category)\nEngine: \(.value.engines)\n\n"'
#+end_src
:END:
