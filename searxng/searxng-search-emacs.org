#+TITLE: searxng search
#+OPTIONS: toc:nil num:nil
#+STARTUP: overview hideblocks 
#+SELECT_TAGS: export
#+PROPERTY: header-args :results output wrap append :noweb yes :var page=1 query="Buster Keaton"
* usage
** global search

edit the PROPERTY at the top of the file and change the query and page number

#+begin_example
#+PROPERTY: header-args :results output wrap append :noweb yes :var page=1 query="buster keaton"
#+end_example

** custom search

You can override the global search using a custom search using header-args

As shown below without the comma before the heading

#+begin_example
,*** custom general search
:PROPERTIES:
:header-args:sh: :var category="general" page=1 query="harold lloyd"
:END:
#+NAME: custom-general-search
#+BEGIN_SRC sh 
<<general-noweb>>
#+end_src
#+end_example

** org refesh

To tell Org-mode to re-scan the entire file for changes to #+PROPERTY put your cursor on the line and press:

#+begin_example
C-c C-c
#+end_example

** export results

1) Move your cursor so it is physically on the * results headline

2) Press C-c C-e to open the export settings
  
3) Press C-s this will toggle the setting from Buffer to Subtree

4) Once it says Subtree, press your export keys (e.g., m m for Markdown).

* noweb code blocks
** searxng-curl noweb
:CODE:
#+NAME: searxng-curl
#+begin_src sh 
curl -s -G "http://localhost:8080/search" \
--data-urlencode "q=${query}" \
--data "format=json" \
--data "categories=${category}" \
--data "pageno=${page}" \
--data "language=auto" \
--data "time_range=" \
--data "safesearch=0"
#+end_src
:END:

** general noweb
:CODE:
#+NAME: general-noweb
#+begin_src sh :noweb yes
<<searxng-curl>> | jq -rj '.results | sort_by(.score) | reverse | to_entries[] | 
"\(.key + 1): [[\(.value.url)][\(.value.title)]]\n" + 
(if (.value.content and .value.content != "") then "\n\(.value.content)\n\n" else "\n" end) +
"Score: \(.value.score)\nCategory: \(.value.category)\nEngine: \(.value.engines)\n\n"'
#+end_src
:END:

** it noweb
:CODE:
#+NAME: it-noweb
#+begin_src sh :noweb yes
<<searxng-curl>> | jq -rj '.results | sort_by(.score) | reverse | to_entries[] | 
"\(.key + 1): [[\(.value.url)][\(.value.title)]]\n" + 
(if (.value.content and .value.content != "") then "\n\(.value.content)\n\n" else "\n" end) +
(if (.value.package_name and .value.package_name != "") then "Name: \(.value.package_name)\n" else "" end) +
(if (.value.version and .value.version != "") then "Version: \(.value.version)\n" else "" end) +
(if (.value.maintainer and .value.maintainer != "") then "Maintainer: \(.value.maintainer)\n" else "" end) +
(if (.value.pubdate and .value.pubdate != "") then "Updated at: \(.value.pubdate)\n" else "" end) +
(if (.value.tags and (.value.tags | length) > 0) then "Tags: \(.value.tags | join(", "))\n" else "" end) +
(if (.value.popularity or .value.stars) then "Popularity: \(.value.popularity // .value.stars)\n" else "" end) +
(if (.value.license_name and .value.license_name != "") then (if .value.license_url then "License: [[\(.value.license_url)][\(.value.license_name)]]\n" else "License: \(.value.license_name)\n" end) else "" end) +
(if (.value.homepage and .value.homepage != "" and .value.homepage != .value.url) then "Docs: [[\(.value.homepage)][Project Homepage]]\n" else "" end) +
(if (.value.source_code_url and .value.source_code_url != "") then "Source: [[\(.value.source_code_url)][View Code]]\n" else "" end) +
"Score: \(.value.score)\nCategory: \(.value.category)\nEngine: \(.value.engines)\n\n"'
#+end_src
:END:
