**** mcp-searxng-enhanced script

#+begin_src sh
#!/bin/sh

# Force unbuffered output so the stream isn't delayed
export PYTHONUNBUFFERED=1

# Filter the output:
# 1. '--line-buffered' ensures Emacs gets the data immediately.
# 2. '^{"jsonrpc"' matches only the start of valid MCP/JSON-RPC messages.
exec /usr/local/bin/doas /usr/local/bin/podman run -i --rm --network=host \
    -e "SEARXNG_URL=http://localhost:8080" \
    -e "PYTHONUNBUFFERED=1" \
    localhost/mcp-searxng-enhanced:local | grep --line-buffered '^{"jsonrpc"'
#+end_src

** search
*** web search
**** Task: Search the web for the "query"

Use 'search_web' for the query "query".
Set the 'categories' parameter to ["general"].

you must extract the following fields

1) url
2) title 
3) content

for every result you must print the result in the following format

and then use the get_website tool to get the website page content and save the web page under - page

- url
- title
- content
- page

***** Mandatory Formatting Rules:
- For each result, print exactly: [[URL][Title]]
- Below each link, print: Snippet: [Content]
- DO NOT use any Markdown formatting.
- DO NOT use #+begin_reasoning.
