** run ollama

To run ollama, plese open 2 terminals.

1. In the first terminal, please run:

#+begin_src sh
OLLAMA_NUM_PARALLEL=1 OLLAMA_DEBUG=1 LLAMA_DEBUG=1 ollama start
#+end_src   

2. In the second terminal pull the model you want to install
