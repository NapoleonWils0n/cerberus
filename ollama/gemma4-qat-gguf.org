** download the gguf files

Download the gguf files

#+begin_src sh
curl -L -o gemma-4-E2B-it-mmproj.gguf 'https://huggingface.co/google/gemma-4-E2B-it-qat-q4_0-gguf/resolve/main/gemma-4-E2B-it-mmproj.gguf'
#+end_src

#+begin_src sh
curl -L -o gemma-4-E2B_q4_0-it.gguf 'https://huggingface.co/google/gemma-4-E2B-it-qat-q4_0-gguf/resolve/main/gemma-4-E2B_q4_0-it.gguf'
#+end_src

** Modelfile
*** Model file with adapter for media

Create the Modelfile in the same directory as the gguf files

#+begin_src conf
FROM ./gemma-4-E2B_q4_0-it.gguf
ADAPTER ./gemma-4-E2B-it-mmproj.gguf

PARAMETER temperature 1.0
PARAMETER top_p 0.95
PARAMETER top_k 64
#+end_src

*** Model file without adapter for media

Create the Modelfile in the same directory as the gguf files

#+begin_src conf
FROM ./gemma-4-E2B_q4_0-it.gguf

PARAMETER temperature 1.0
PARAMETER top_p 0.95
PARAMETER top_k 64
#+end_src

** create the model

#+begin_src sh
ollama create gemma4-qat -f Modelfile
#+end_src
