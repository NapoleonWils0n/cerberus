#+STARTUP: content
* fdkaac build freebsd 
** resources

[[https://github.com/nu774/fdkaac]]

** packages

Install the required packages

#+begin_src sh
doas pkg install git autoconf automake libtool pkgconf fdk-aac
#+end_src

** clone the git repository

Clone the fdkaac git repository

#+begin_src sh
git clone https://github.com/nu774/fdkaac.git
#+end_src

** change into the git repository

#+begin_src sh
cd fdkaac
#+end_src

** create the configure file

#+begin_src sh
autoreconf -i
#+end_src

** configure and make

Run the configure script

#+begin_src sh
./configure
#+end_src

make

#+begin_src sh
make
#+end_src

** copy the fdkaac binary into your bin directory

#+begin_src sh
cp fdkaac ~/bin
#+end_src

** fdkaac help

Show the fdkaac help

#+begin_src sh
fdkaac -h
#+end_src
