** Create a Local Git Branch for Your Customizations

First, navigate to your Poudriere ports tree directory. Based on your configuration, this is located where Poudriere cloned your local ports tree.  

#+begin_src sh
cd /usr/local/poudriere/ports/local
#+end_src

Now, create and switch to a new branch (let's call it ardour-patches) based on your active quarterly branch:

#+begin_src sh
doas git checkout -b ardour-patches
#+end_src

** Download and Apply the Patches

#+begin_src sh
doas fetch -o patch1.diff "https://bz-attachments.freebsd.org/attachment.cgi?id=267715"
doas patch -p1 < patch1.diff
doas rm patch1.diff 
#+end_src

#+begin_src sh
doas fetch -o audio/ardour/files/patch-libs_ardour_port__manager.cc "https://bz-attachments.freebsd.org/attachment.cgi?id=267716"
doas fetch -o audio/ardour/files/patch-libs_ardour_control__protocol__manager.cc "https://bz-attachments.freebsd.org/attachment.cgi?id=267717"
#+end_src

** Commit Your Patches Locally

Commit these modifications to your local ardour-patches branch.

This creates a safe snapshot that Git can track and rebase later.

#+begin_src sh
doas git add .
doas git commit -m "audio/ardour: Apply Bugzilla 289714 patches"
#+end_src

** How to Handle Future Updates (The Rebase Strategy)

When 2026Q2 gets official upstream updates, you won't be able to just run poudriere ports -u -p local  directly because you are on a custom branch. Instead, update your ports tree using this clean Git workflow:  

#+begin_src sh
cd /usr/local/poudriere/ports/local
#+end_src

To update your ports tree to the next quarterly branch, type:

#+BEGIN_SRC sh
doas poudriere ports -u -p local -B 2026Q3
#+END_SRC

Switch back to your custom branch and replay your patches on top of the new updates

#+begin_src sh
doas git checkout ardour-patches
doas git rebase 2026Q3
#+end_src

** login_conf

Memory locking has to be allowed in /etc/login.conf or ~/.login_conf. Set the
resource limit ":memorylocked=unlimited:" and don't forget to run

#+begin_src sh
vi ~/.login_conf
#+end_src

#+begin_src conf
me:\
  :charset=UTF-8:\
  :lang=en_GB.UTF-8:\
  :setenv=LC_COLLATE=C:\
  :memorylocked=unlimited:
#+end_src

#+begin_src sh
cap_mkdb ~/.login_conf
#+end_src
