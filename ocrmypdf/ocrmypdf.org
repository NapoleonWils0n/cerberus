** ocrmypdf usage

#+begin_src sh
ocrmypdf \
--sidecar extracted_text.txt \
--force-ocr \
--output-type pdf \
input.pdf \
output.pdf
#+end_src

** format text

the extracted text width is quite narrow so we use the fmt command to increase the width

#+begin_src sh
fmt -w 100 extracted_text.txt > output.txt
#+end_src
